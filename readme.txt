Instructions for setting up the project and running the code. 

Note that python3 is expected, but python3.7 is the newest version that supports the scikit-image dependency (at time of writing). It is suggested to use pyenv for python management with venv for the python/pip package installation management.

************************************************
Python Package Installation
************************************************
Set up virtual environment:
	python3 -m venv env
Activate the virtual environment:
	source env/bin/activate
Install the required packages in the virtualenv using pip:
	pip install -r requirements.txt
